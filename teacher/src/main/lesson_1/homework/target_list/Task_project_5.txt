Перепешите методы удаления и изменения данных на работу с id.

Пользователь указывает id цели, которую хочет удалить,
программа находит и удаляет из списка цель с указанным id.

Пользователь указывает id цели, у которой хочет изменить
название или описание, программа находит в списке цель с указанным id
и меняет у неё свойства.
