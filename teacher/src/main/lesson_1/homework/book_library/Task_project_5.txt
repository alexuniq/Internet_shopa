Перепешите методы удаления и изменения данных на работу с id.

Пользователь указывает id книги, которую хочет удалить,
программа находит и удаляет из списка книгу с указанным id.

Пользователь указывает id книги, у которой хочет изменить
название или автора, программа находит в списке книгу с указанным id
и меняет у неё свойства.
