Ваша первая задача на данном этапе это создание маленького
консольного приложения прототипа с очень маленьким функционалом.
Это будет ваша отправная точка.

Предлагается создать простое консольное приложение по управлению
списком товаров со следующей функциональностью:
- добавить товар в список,
- удалить товар из списка,
- распечатка список товаров на консоль.

Для этого создайте класс ProductListApplication с main() методом.

Ваше приложение должно выводить доступные пользователю пункты
меню на консоль, потом запрашивать у пользователя какой из
пунктов меню пользователь хочет выполнить, выполнять выбранный
пункт меню и снова показывать меню программы пользователю. И
так до тех пор пока пользователь не выберет пункт меню завершения
работы программы.

На данном этапе создаваемые объекты храните в памяти программы,
используйте для этого например структуру данных список.

Задачу можно разделить между членами команды,
каждый реализует свой пункт меню.

Если необходимо вы можете расширить функциональность вашего приложения
добавив например поиск товара по названию, изменение
названия или описания уже добавленного товара. Реализацию каждого
нового пункта меню можно оформить отдельной задачей в Kanban Board
и назначить ответственного за ее выполнение.
